## 3. 实现范围

- 用户可使用邀请码注册，注册后默认为 `active` member。
- 用户可使用用户名或邮箱登录，认证方式为 DRF Token。
- 已登录用户可查看当前用户、退出登录、访问测试用例列表。
- admin 可创建、分页查看、禁用邀请码；member 不显示邀请码管理入口，直达管理页会被拦截。
- admin 可同步 `api-test/test_case/` 下 pytest 用例元数据；member 只能查看列表。
- 测试用例列表展示包名、模块、路径、类名、函数名、标题/描述、node id、同步状态和更新时间，支持分页、关键字、包名、模块筛选。
