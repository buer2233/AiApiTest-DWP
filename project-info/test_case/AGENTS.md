# test_case AGENTS.md

本目录用于存放项目整体测试用例资料，包括验收用例、功能测试用例、接口测试场景、端到端流程用例和回归测试清单。

## 强制技能

- 进入本目录编写或修改测试用例资料时，必须使用 `/test-cases` 技能。
- 该技能来源于 `https://github.com/stellarlinkco/myclaude`。
- 开始工作前必须检查当前环境是否可用该技能。
- 如果检查到该技能无法使用，应推荐用户先安装 `/test-cases` 技能后再继续，不要直接跳过测试用例生成技能流程。

## 文件约定

- 测试用例文档使用简体中文，专业名词可保留英文原名。
- 推荐文件名：`测试用例.md`、`验收测试用例.md`、`回归测试清单.md`。
- 本目录只放测试设计资料，不放 `api-test/` 中可执行 pytest 用例代码。

## 内容要求

- 每条用例应包含用例编号、测试目标、前置条件、操作步骤、预期结果和优先级。
- 测试场景应覆盖登录、任务执行、失败重试、模块重试、Jenkins 查询、Allure 报告入口和权限边界。
