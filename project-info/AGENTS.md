# project-info AGENTS.md

本目录是项目说明资料统筹目录，只负责沉淀需求、原型、测试用例、架构图和流程图等项目交接资料，不存放业务实现代码或运行产物。

## 子目录职责

| 目录 | 用途 |
|------|------|
| `demand/` | 存放项目整体需求分析、需求规格、业务流程和验收口径。 |
| `UI/` | 存放项目整体 UI 原型图、页面交互说明和原型生成提示词。 |
| `test_case/` | 存放项目整体测试用例、验收用例和测试场景设计资料。 |
| `project_picture/` | 存放项目架构图、流程图、项目说明图片和架构说明文档。 |

## 通用规则

- 修改本目录前先读取根目录 `AGENTS.md` 和主计划文件。
- 资料必须使用简体中文编写，专业名词可保留英文原名。
