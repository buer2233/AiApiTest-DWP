# UI AGENTS.md

本目录用于存放项目整体 UI 原型资料，包括页面原型图、交互说明、组件布局、页面状态和原型生成提示词。

## 强制技能

- 进入本目录编写或修改 UI 原型资料时，必须使用 `prototype-prompt-generator` 技能。
- 该技能来源于 `https://github.com/stellarlinkco/myclaude`。
- 开始工作前必须检查当前环境是否可用该技能。
- 如果检查到该技能无法使用，应推荐用户先安装 `prototype-prompt-generator` 技能后再继续，不要直接跳过原型生成技能流程。

## 文件约定

- UI 文档使用简体中文，专业名词可保留英文原名。
- 推荐文件名：`UI原型说明.md`、`页面交互说明.md`、`prototype-prompts.md`。
- 原型图文件应使用清晰命名，例如 `login-page.png`、`module-pass-rate-page.png`、`failure-cases-dialog.png`。

## 内容要求

- UI 原型应覆盖登录页、平台布局、模块通过率页、失败用例弹窗、重试操作、Jenkins 任务入口和 Allure 报告入口。
- 说明应包含页面目标、主要组件、交互状态、空状态、错误状态和响应式要求。
- 不提交真实账号、密码、token、cookie、生产地址或公司专有业务字段。
