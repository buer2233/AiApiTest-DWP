## 2. 设计原则

- 企业级管理后台：安静、清晰、可扫描，优先表单效率、表格密度和状态反馈。
- 视觉语言：温暖中性色背景、炭黑/深灰正文、暖陶土色主强调、浅边框、克制阴影、8px 以内圆角。
- 不做营销式落地页，不使用装饰性插画、渐变大背景或卡片堆叠。
- 数据密集页面保持紧凑：筛选、表格、分页、批量/单项操作必须稳定且不跳动。

## 4. 页面结构

### 4.1 登录页 `/login`

```text
登录页
├─ 左侧品牌区（桌面显示，移动端折叠）
│  ├─ 平台名 AiApiTest-DWP
│  └─ 简短说明：AI 协作自动化测试平台
└─ 登录表单
   ├─ 账号输入框（用户名或邮箱）
   ├─ 密码输入框
   ├─ 登录按钮
   ├─ 错误提示区
   └─ 注册入口链接
```

- **正常态**：账号、密码合法时可提交。
- **提交中**：登录按钮 loading，输入框可保持只读。
- **错误态**：统一提示“账号或密码不正确”，不暴露账号是否存在。
- **成功态**：跳转 `/test-cases`。

### 4.2 注册页 `/register`

```text
注册页
├─ 注册表单
│  ├─ 邀请码输入框
│  ├─ 用户名输入框
│  ├─ 邮箱输入框
│  ├─ 密码输入框
│  ├─ 确认密码输入框
│  ├─ 注册按钮
│  └─ 返回登录链接
└─ 字段级错误提示
```

- **邀请码错误**：在邀请码输入框下显示“邀请码无效或不可用”。
- **密码错误**：密码长度不足或两次不一致时显示字段级错误。
- **成功态**：展示“注册成功，请登录”，随后跳转 `/login`。

### 4.3 邀请码管理页 `/invite-codes`

```text
邀请码管理页（仅 admin）
├─ 顶部栏
│  ├─ 页面标题
│  └─ 创建邀请码按钮
├─ 筛选区
│  ├─ 关键字输入
│  └─ 状态选择
├─ 邀请码表格
│  ├─ code
│  ├─ 状态
│  ├─ 最大使用次数
│  ├─ 已使用次数
│  ├─ 过期时间
│  ├─ 创建人
│  └─ 操作：禁用
├─ 分页器
└─ 创建邀请码弹窗
   ├─ code
   ├─ max_uses
   ├─ expires_at
   └─ 提交 / 取消
```

- **权限态**：member 不显示菜单入口；直接访问时显示 403 或跳回测试用例页。
- **创建成功**：刷新列表，toast 展示“邀请码已创建”。
- **禁用操作**：二次确认；成功后状态标签变为 `disabled`。
- **空状态**：展示“暂无邀请码”。

### 4.4 测试用例列表页 `/test-cases`

```text
测试用例列表页
├─ 顶部栏
│  ├─ 页面标题
│  ├─ 当前用户
│  └─ 退出登录
├─ 筛选区
│  ├─ 关键字输入
│  ├─ 包名输入或选择
│  ├─ 模块输入或选择
│  └─ 查询 / 重置
├─ 操作区
│  └─ 同步测试用例按钮（仅 admin）
├─ 测试用例表格
│  ├─ node id
│  ├─ 包名
│  ├─ 模块
│  ├─ 文件路径
│  ├─ 类名
│  ├─ 函数名
│  ├─ 标题
│  ├─ 描述
│  └─ 同步时间
└─ 分页器
```

- **加载态**：表格骨架屏或 Element Plus loading。
- **空状态**：无数据时显示“暂无测试用例”，筛选无结果时显示“没有匹配的测试用例”。
- **错误态**：展示错误条和“重试”按钮。
- **同步反馈**：admin 点击同步后显示 loading；成功提示新增、更新、总数。
- **member 权限态**：不展示同步按钮。
- **未登录态**：路由守卫跳转 `/login`。

## 5. 字段来源

| UI 字段 | API 字段 | 页面 |
| --- | --- | --- |
| 当前用户 | `GET /api/v1/auth/me` | 顶部栏 |
| 邀请码 code | `GET /api/v1/invite-codes[].code` | 邀请码管理 |
| 邀请码状态 | `GET /api/v1/invite-codes[].status` | 邀请码管理 |
| 最大/已用次数 | `max_uses` / `used_count` | 邀请码管理 |
| 测试用例 node id | `GET /api/v1/test-cases[].node_id` | 测试用例列表 |
| 包名 | `package_name` | 测试用例列表 |
| 模块 | `module_name` | 测试用例列表 |
| 文件路径 | `file_path` | 测试用例列表 |
| 标题/描述 | `title` / `description` | 测试用例列表 |

## 6. 覆盖校准

| 测试类别 | 覆盖结论 |
| --- | --- |
| 正常场景 | 登录、注册、邀请码创建/列表/禁用、用例同步、用例列表均有页面状态 |
| 异常场景 | 登录失败、注册字段错误、邀请码错误、API 失败、权限拒绝均有反馈 |
| 边界值 | 密码长度、邀请码长度、max_uses、空列表、无匹配筛选均已覆盖 |
| 权限态 | 未登录跳转；admin/member 菜单和按钮差异已覆盖 |
| 状态流转 | 邀请码 active/disabled/exhausted/expired；Token 退出失效；用例 synced/missing 已覆盖 |
| 容器化 | 前端只调用相对 API 或环境变量 API，不拼接本机路径 |
